Installation

To install SUBSIM simply execute the following command lines in STATA:

set more off

net from https://subsim.vercel.app/subsim/Installer

net install subsim_part1, force

net install subsim_part2, force

cap addSMenu profile.do _subsim_menu

Once the package is installed, use the SUBSIM package manager for updates and for consulting this website.

 

Important Note

The last Stata command line tries to add the file profile.do automatically or add the command _subsim_menu in the file profile.do if the latter exists already. If this last command does not function, you have to copy the profile.do file as indicated below:

  1. Windows  OS system: copy the file in c:/ado/personal/

  2. Macintosh system: copy the file in one of the Stata system directories. To find these directories, type the command sysdir.
